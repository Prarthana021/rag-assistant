# Personal Knowledge Base

Markdown files in this directory are the source of truth for retrieval and
generation. Qdrant contains only a derived search index that can be rebuilt.

Do not add private personal content to this public repository. Before filling
these templates, choose a privacy strategy such as keeping the populated data
directory outside Git, using a private repository, or maintaining sanitized
public examples separately.

## Writing guidelines

- Use descriptive headings so sections can be chunked independently.
- Record concrete facts and metrics only when they are accurate.
- Keep one experience, story, project, or concept per section where practical.
- Add role or technology keywords naturally in the relevant section.
- Edit these files directly when the source information changes.
